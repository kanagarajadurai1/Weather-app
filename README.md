# 🌦️ WeatherNow — Real-Time Weather Application

A sleek and responsive weather application built with **React.js**, delivering real-time weather insights for cities worldwide. This project highlights efficient API integration, clean component architecture, and a modern user interface.

---

## 🚀 Features

- 🔍 Instant city-based weather search  
- 🌡️ Live temperature, humidity, wind speed & pressure  
- 🌍 Global weather data integration  
- ⚡ Fast and responsive performance  
- 🎨 Modern glassmorphism-inspired UI  
- ⌨️ Keyboard-friendly (Enter key search)  
- ❌ Robust error handling for invalid inputs  

---

## 🛠️ Tech Stack

- ⚛️ React.js (Hooks & Functional Components)  
- 🌐 Weather API Integration  
- 🎨 Custom CSS Styling  
- 📦 JavaScript (ES6+)  


---

## 📌 Key Highlights

- 📊 Efficient state management using React Hooks  
- 🧩 Modular and reusable component structure  
- 🌐 Real-world API integration  
- 🎯 Clean and intuitive user experience  
- ⚠️ Graceful error handling system
